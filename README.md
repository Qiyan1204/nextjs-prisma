## Uptime Monitoring

This project includes SLA-style uptime sampling:

- `GET /api/health` performs a lightweight DB probe and records `health_ok` or `health_fail` telemetry.
- `vercel.json` schedules this endpoint every minute using Vercel Cron.
- Homepage `Uptime` percentage is computed from these success/failure samples over the recent 24h window.

Notes:

- Vercel Cron runs only on Vercel deployments.
- In local development, call `/api/health` manually or with your own scheduler if you want continuous samples.
